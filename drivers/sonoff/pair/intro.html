Before a Sonoff device can be added, it should be configured to use Homey its server.

For more information about how to set up your device, click here.

During the next step, Homey will start looking for Sonoff devices. This works best when the device is plugged in after Homey started scanning.

Supported devices:

  • Sonoff Basic WiFi Wireless Switch
  • Sonoff S20 Smart Socket
  • Sonoff RF WiFi Wireless Smart Switch With RF Receiver
  • Sonoff Slampher

Voordat een Sonoff apparaat toegevoegd kan worden moet het zo ingesteld zijn dat het de Homey als server gebruikt.

Voor meer informatie over het configureren van het apparaat, klik hier.

Bij de volgende stap zal Homey op zoek gaan naar Sonoff apparaten. Dit werkt het beste als het apparaat pas ingeplugd wordt als Homey aan het zoeken is.

Ondersteunde apparaten:

  • Sonoff Basic WiFi Wireless Switch
  • Sonoff S20 Smart Socket
  • Sonoff RF WiFi Wireless Smart Switch With RF Receiver
  • Sonoff Slampher
